# 网上商城管理系统 java1178

## 项目概述

本项目是基于SSM框架的网上商城管理系统，包含源代码、所需环境、数据库及15000字的详尽文档。系统具备购物车、商品分类、前台下单、后台管理等丰富功能，适合作为毕业设计的选题。

## 技术栈

- Java
- Spring
- SSM（SpringMVC、Spring、MyBatis）
- JSP
- jQuery

## 功能模块

- 购物车
- 商品分类
- 前台下单
- 后台管理
- 登录注册

## 系统角色

- 用户端
- 管理员端

- 数据库：MySQL 5.1 及以上（包含8、9版本）
- JDK版本：1.8 至 17以下
